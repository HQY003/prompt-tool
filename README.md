# 场景化提示词工具 · Prompt Lab

本地运行的场景化 AI 提示词库，覆盖萌宠 IP、视频脚本、电商生图、产品经理、备考学习等方向。选择场景 → 填写变量 → 一键复制，粘贴到 ChatGPT / Claude / Cursor 即可使用。

## 快速开始

直接用浏览器打开 `index.html` 即可，无需安装依赖。

```bash
# Windows
start index.html

# 或双击 index.html
```

如需本地服务器（可选）：

```bash
cd prompt-tool
npx serve .
# 访问 http://localhost:3000
```

## 功能

- **5 大分类、28+ 场景**：每个场景都是可直接使用的案例提示词
- **变量填充**：填写产品名、主题等，实时预览替换后的提示词
- **追问示例**：每个场景 5 条追问，首轮回答后一键复制继续深挖
- **一键复制**：复制完整提示词到剪贴板
- **搜索**：按关键词、标签搜索场景
- **纯本地**：数据在本地，不上传服务器

## 自定义扩展

编辑 `data/prompts.js` 即可新增分类或场景，结构如下：

```javascript
{
  id: 'unique-id',
  title: '场景标题',
  description: '简短说明',
  tags: ['标签1', '标签2'],
  variables: [
    { key: 'var_name', label: '变量名', placeholder: '填写示例' }
  ],
  prompt: `提示词正文，用 {{var_name}} 引用变量`
}
```

## 使用建议

1. 先选最接近的场景，复制基础提示词
2. 在 AI 对话中根据首轮输出继续追问细节
3. 同一提示词可追加约束，如「输出表格」「控制在 500 字内」
4. 萌宠 IP、视频脚本类建议多轮迭代，先框架后细节
